Enhancements

Hover Effect

The hover effect over the group members in the footer is a design effect where when the users mouse is hovered over a certain element, it creates an animation. On the home page when the user hovers over the group members photos, it expands and changes the colour of the image and displays the members name and student ID. This goes beyond the basic requirements as it uses pseudo classes, image-container and transitions to create this effect.

We implemented the hover effect on the index.html page linked here: Index We used this video and code found online to help code the effect: Hover Effect Video

Trail Effect

The text trail effect is a design where the text is printed several times but each time it is slightly below the last text. Each text is on a timer to appear and then disappear to create the bouncing effect. It is another optical illusion as it looks as if there is one printed text moving up and down. Each text is a different colour to create a rainbow effect.
